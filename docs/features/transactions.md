## Overview

The Transactions page is your central hub for managing all financial activity. View, filter, edit, and analyze your income and expenses.

---

## Adding Transactions

### Single Transaction

1. Click **Add Transaction**
2. Fill in:
   - **Amount** - Transaction value
   - **Description** - What it's for
   - **Category** - Income, expense, or transfer category
   - **Account** - Source/destination account
   - **Date** - When it occurred
   - **Tags** (optional) - For cross-category tracking
3. Click **Save**

### Transfers

For moving money between accounts:

1. Click **Add Transfer**
2. Select **From** and **To** accounts
3. Enter amount and date
4. Click **Save**

This creates linked transactions in both accounts.

## Filtering Transactions

Use the filter panel to narrow down transactions:

| Filter | Options |
|:-------|:--------|
| **Date Range** | Custom dates, month, or year presets |
| **Categories** | Single or multiple categories |
| **Accounts** | Specific accounts |
| **Account Type** | Linked (bank) or Manual |
| **Tags** | Filter by tag |
| **Status** | Bank reconciled, not reconciled |
| **Search** | Text search in descriptions |

{: .note }
> Filters are preserved in the URL for easy bookmarking and sharing.

---

## Bulk Editing

Edit multiple transactions at once:

1. **Select transactions** using checkboxes
2. Click **Bulk Edit**
3. Choose what to change:
   - Category
   - Account
   - Date
   - Tags
4. Apply changes

## Staging Area

Bank-synced transactions appear in staging first:

1. Review pending transactions
2. **Accept** to add to your ledger
3. **Dismiss** to ignore
4. Bulk actions available for efficiency
