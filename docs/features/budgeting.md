## Recurring Transactions

### What Are Recurring Transactions?

Recurring transactions are expected expenses that happen regularly:

- Rent/mortgage
- Subscriptions (Netflix, Spotify)
- Insurance premiums
- Utility bills

### Setting Up Recurring Expenses

1. Navigate to **Budget** page
2. Find the **Recurring Expenses** section
3. Click **Add Recurring**
4. Fill in:
   - Description
   - Amount
   - Category
   - Frequency (monthly, weekly, etc.)
   - Start date
5. Click **Save**

### Missing Transaction Alerts

When a recurring transaction is overdue:

- Dashboard shows a warning indicator
- Click to see details:
  - Transaction name
  - Days overdue
  - Expected amount
- Helps catch missed bills before they become problems

---

## Planned Expenses

For one-time future expenses:

1. Go to **Budget** page
2. Find **Planned Expenses** section
3. Click **Add Planned**
4. Enter description, amount, category, and expected date
5. Track against actual when it occurs
