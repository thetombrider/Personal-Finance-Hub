## Category Management

### Creating Categories

1. Go to **Settings** → **Categories**
2. Click **Add Category**
3. Configure:
   - **Name** - Category name
   - **Type** - Income, Expense, or Transfer
   - **Color** - Visual identification
   - **Icon** - Optional icon
   - **Monthly Budget** - Budget target (expense categories)
4. Click **Save**

### Category Types

| Type | Purpose | Effect on Net Worth |
|:-----|:--------|:--------------------|
| **Income** | Money coming in | Increases |
| **Expense** | Money going out | Decreases |
| **Transfer** | Movement between accounts | Neutral |

## Tag Management

Tags offer flexible organization across categories:

1. Go to **Settings** → **Tags**
2. Click **Add Tag**
3. Set name and color
4. Use tags on any transaction

Examples: "Vacation", "Tax Deductible", "Business Expense"

## Font Customization

Change the application font:

1. Go to **Settings**
2. Find **Appearance** section
3. Select from available fonts (Inter, Roboto, Outfit, etc.)
4. Changes apply immediately
