## Overview

Track your investment portfolio with real-time stock quotes, trade history, and performance metrics.

---

## Holdings View

The main Portfolio page displays:

| Column | Description |
|:-------|:------------|
| **Symbol** | Stock ticker |
| **Shares** | Number of shares owned |
| **Avg Cost** | Average purchase price |
| **Current Price** | Real-time quote (Yahoo Finance) |
| **Market Value** | Current total value |
| **Gain/Loss** | Unrealized profit/loss |
| **% Change** | Percentage return |

---

## Adding Trades

### Record a Trade

1. Click **Add Trade**
2. Fill in:
   - **Symbol** - Stock ticker (e.g., AAPL)
   - **Type** - Buy or Sell
   - **Shares** - Number of shares
   - **Price** - Price per share
   - **Date** - Trade date
   - **Account** - Investment account
3. Click **Save**

Holdings are automatically updated based on your trades.

---

## Trade History

View all historical trades:

- Paginated list with 50 items per page
- Sort by date, symbol, or amount
- Filter by account
- Click to edit or delete

---

## Market Data

Real-time quotes powered by Yahoo Finance:

- Prices refresh on page load
- Click the refresh icon to update
- Delayed quotes (15-20 minutes typical)
